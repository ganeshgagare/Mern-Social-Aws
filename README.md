# Mern-Social-Aws

Platform Used : React, MongoDB Database, Node.js, .NET, AWS Deployment

Developed a social media mini web application that allows users to create accounts, upload photos, follow other users, and like or comment on posts. Implemented secure authentication, profile management, and personalized feeds using a React-based frontend and a microservices backend. The application uses MongoDB for efficient data handling and is deployed on AWS, ensuring scalability, smooth performance, and a clean user experience.
